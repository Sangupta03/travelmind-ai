What It Does

Built for Smart Travel

Not just an AI chatbot — a full multi-agent reasoning system that thinks like a travel expert team.

Multi-Agent Architecture

5 specialized agents each with a distinct role — Budget, Comfort, Experience, Negotiator, Manager — coordinated by a ManagerAgent pipeline.

Google Maps Routing

Real Distance Matrix API integration with nearest-neighbor ordering and automatic walk-vs-cab decisions for every journey leg.

Budget Optimization

Hard constraint engine enforces your budget ceiling. Live hotel pricing via SerpAPI. Cost breakdown across flights, hotel, transport, and food.

Parallel Execution

Budget, Comfort, and Experience agents run simultaneously using ThreadPoolExecutor — reducing processing time by ~60% vs sequential.

Secure Authentication

bcrypt password hashing, JWT tokens, httponly cookies. Your account and trips are fully persisted in SQLite via SQLAlchemy ORM.

User Memory

UserAgent extracts and stores your constraints — walking preference, dietary needs, travel companions — and applies them automatically.
